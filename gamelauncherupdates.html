Reflex Interactive Game Launcher

Version 0.1.0

- Initial Release

The first public release of the Game Launcher is now available! This version introduces the core functionality, allowing users to create accounts and store their games locally. The launcher is designed to provide a smooth and modern experience for managing your game library. Below is a detailed breakdown of the key features available in this version:

ðŸ”¹ User Accounts & Authentication

  • Users can create an account and log in to personalize their game collection.
  • Login button updates to display the username upon signing in.
  • Once logged in, the button navigates to the account page.
  • A logout button is available in the navbar for easy account management.

ðŸ”¹ Store Page

  • Displays a dynamic list of games available for purchase.
  • Clicking on a game opens its detailed Game Details page.
  • Each game listing includes a title, cover image, price, genres, tags, and system requirements.
  • A "Buy Game" button allows users to add purchased games to their library.
  • The purchase button updates correctly depending on whether the game is owned.

ðŸ”¹ Game Details Page

  • Displays in-depth information about a selected game.
  • Includes a large game screenshot, title, description, genres, and tags.
  • System requirements are listed for users to check compatibility.
  • Features a "Buy Game" button that updates dynamically after purchase.
  • Users can add the game to their Wishlist for later consideration.
  • A "Back" button in the top left allows easy navigation back to the Store.

ðŸ”¹ Library Page

  • Owned games are stored and displayed dynamically in the user's library.
  • Each game listing includes its title and cover image.
  • Games can be launched directly from the library (future update for actual launching).
  • The UI is designed for a sleek, dark theme consistent across all pages.

ðŸ”¹ Wishlist Feature

  • A dedicated Wishlist tab is available in both the Store and Library pages.
  • Users can add games to their Wishlist from the Game Details page.
  • Wishlisted games are displayed dynamically, making it easy to track desired purchases.

ðŸ”¹ UI & Design

  • The launcher features a dark, modern, and consistent theme across all pages.
  • Navigation is simple and intuitive, with easy access to Store, Library, and Wishlist sections.
